# Sales-Revenue-Analysis-Dashboard
Excel-based Sales &amp; Revenue Analysis Dashboard with KPI tracking and interactive business insights.


## Objective
To analyze sales and revenue performance using Excel dashboards.

## Features
- Total Sales KPI
- Total Profit KPI
- Total Quantity KPI
- Sales by Category Analysis
- Sales by Region Analysis
- Top Sub-Categories by Sales

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
